brag

This application provides the user a storage of all the NFTs. User gets to add any NFT they want from other websites by proving the address and a token of that NFT. By clicking on an added NFT, user can edit and delete the NFT. Home page has a log in and a sign up button that takes the user to those pages. Home page also has a carousel of NFTs.
